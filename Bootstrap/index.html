Bootstrap Overview
  • Popular front-end framework for building responsive and mobile-first websites & applications.
  • Bundle of CSS classes (properties + values) available via CDN or downloadable files.
  • Predefined codes: built-in classes, IDs, and JavaScript components.
  • Includes some extra attributes for customization.

ðŸ”¥ Interview-Based Q&A
  • Q1: What is Bootstrap?
    Ans: Bootstrap is a CSS + JS framework to build responsive, mobile-first UI faster using pre-defined classes.
  • Q2: Difference between CDN and Downloaded Bootstrap?
    Ans: CDN = lightweight, fast but requires internet. Download = offline + more customization.
  • Q3: How is Bootstrap different from CSS?
    Ans: CSS is manual styling, Bootstrap gives pre-built utility classes (faster development).
  • Q4: Why Bootstrap is mobile-first?
    Ans: It uses responsive grid system with breakpoints (xs, sm, md, lg, xl).
